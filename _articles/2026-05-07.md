## The decisions

### ADR-0068 enforcement is formally suspended pending suggestion-log audit — *model-behavior-as-design*

ADR-0068 mandates that the agent must follow skill suggestions — specifically, invoking the suggested skill within the same session in which the suggestion was made. The telemetry funnel (4 suggested → 0 followed Wednesday, compounding from Monday's 46 → 0; Tuesday's data is absent from the log and treated as missing, not zero) is contaminated by a known instrumentation issue: comment-marker credits. A comment-marker credit is logged when the agent adds a PR comment containing a skill invocation marker, and the suggestion-log pipeline may be counting those markers as real-time suggestions the agent could have acted on in the current session — which they are not. Until the suggestion log is audited and comment-marker credits are separated from real-time credits, the 0% follow rate cannot be attributed to agent non-compliance versus instrumentation error. ADR-0068 enforcement status is therefore explicitly SUSPENDED — not questionable, not deferred — pending the audit. Treating it as enforceable on synthetic data compounds the problem by optimizing against the wrong signal. If the audit returns clean data and the follow rate is still 0%, that is a confirmed behavioral failure and will be treated as such. The prior on that outcome is non-trivial; it needs to be tested, not assumed away.
